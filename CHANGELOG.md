## 0.1.0

This is the initial version of the package, supporting Webflow v2 API for the following read-only operations:

#### Sites

- List Sites
- Get Site
- List Custom Domains

#### Pages

- List Pages
- Get Page Metadata
- Get Page Content

#### Assets

- List Assets
- Get Asset

#### Collections

- List Collections
- Get Collection
- List Collection Items (all/live)
- Get Collection Item (all/live)
